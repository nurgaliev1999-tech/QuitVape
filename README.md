# QuitVape

PWA-счётчик отказа от одноразок. Работает полностью локально, без сервера и аккаунта — все данные в localStorage браузера/приложения.

## Как открыть и проверить на компьютере

Двойным кликом файлы не откроются как PWA (нужен http, не file://). Проще всего:

1. Открой командную строку в этой папке.
2. `python -m http.server 8124`
3. Открой в браузере `http://localhost:8124`

## Как установить на iPhone

1. Выложи папку на любой https-хостинг (например, GitHub Pages, Netlify, Vercel — бесплатно и без сервера с твоей стороны) или подними её на своём компьютере и открой по локальному IP в Safari на iPhone (если телефон и компьютер в одной сети).
2. Открой адрес в Safari на iPhone.
3. Кнопка «Поделиться» → «На экран Домой».
4. Дальше открывается как обычное приложение, в полноэкранном режиме.

## Данные и резервное копирование

Все данные (счётчик, дневник причин) хранятся только на телефоне. Что сделано для их сохранности:

- Приложение запрашивает у iOS «постоянное хранилище», чтобы Safari не вычищал данные.
- Данные автоматически дублируются в два независимых хранилища на телефоне (localStorage + IndexedDB). Если одно вычистится — при следующем запуске восстановится из второго.
- Раздел «Настройки и данные» на главном экране: кнопка «Сохранить в файл» делает резервную копию (JSON), «Восстановить из файла» — загружает её обратно. Это единственный способ перенести данные на другой телефон, поэтому изредка делай такую копию.

PWA не может сохранять файл автоматически по расписанию — iOS требует подтверждения на каждое сохранение. Поэтому файл-копия делается вручную, а автоматическая защита — это дублирование хранилищ выше.

### Точка отсчёта

По умолчанию счётчик стартует с момента первого открытия. Если бросил раньше — задай реальную дату/время последнего использования в «Настройки и данные» → «Точка отсчёта».

## Структура

- `index.html` — разметка трёх экранов (Главная / Достижения / Ретроспектива) + два модальных окна (тяга, дневник причин)
- `css/style.css` — стили, тёмная тема
- `js/storage.js` — чтение/запись состояния в localStorage
- `js/techniques.js` — список техник преодоления тяги
- `js/achievements.js` — вехи по дням
- `js/retrospective.js` — группировка записей дневника по времени суток / дню недели / причине
- `js/app.js` — вся логика экрана и таймеров
- `manifest.json`, `service-worker.js`, `icons/` — PWA-обвязка (иконка, офлайн-кеш, установка на экран Домой)

## Что дальше можно донастроить

- Иконка сейчас простая, сгенерированная — можно заменить `icons/*.png` на свою.
- Ставка сэкономленных денег зашита в `js/app.js` (`DISPOSABLE_PRICE = 2000`, `DISPOSABLE_DAYS = 14`) — поменять, если цена или частота смены одноразки изменится.
